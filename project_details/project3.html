Project Information

  • Categories: Natural Language Processing, Neural Machine Translation, Active Learing

This is a project I am currenlty working on with the aim to have it (hopefully) published in ACL. This is a part of our research project for a class Universal Natural Language Understading
The goal is to see when working with transformer based NMT taks how does selecting different layers during Active Learning affect the quality of the translations generated by the models using linguistic errors as a metric.
We are using MarianMT as our base NMT model and using English as our target language, we will also run our observations on low resrouce languages like Gujarati. We are follwing active learning methods based off this paper.
We are using a grammatical error correction method based off something Grammarly does.But this is a work in progress and we are working on multiple things, you can find some of the project progress here.
